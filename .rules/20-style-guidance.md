# Style & Design System Guidance
- **Source of Truth**: The [EverGray Tech Design System](https://github.com/EverGrayTech/design-system) owns all foundational visual rules.
- **Website Expression**: This site is the slightly more cinematic expression of the shared system — atmospheric treatments, scroll-reveal, and marketing composition are permitted within upstream boundaries.
- **Atmosphere**: Gradient fields ≤0.15 opacity, radial highlights 0.06–0.12 opacity. No glassmorphism, neon glow, gradient text, colored shadows, or bokeh/lens effects.
- **Motion**: Section reveals 250–400ms / 16–24px translateY. Hero ambient 30s+ cycles. Hover transitions 150ms. All disabled under `prefers-reduced-motion`.
- **CTA Hierarchy**: Primary uses accent treatment (reserved for true emphasis). Secondary uses neutral outline. Accent is not the default action styling.
- **Contrast**: Marketing headlines target 7:1+ contrast. Test at the lightest gradient point, not the average.
- **Prohibited**: Glassmorphism, neon glow, gradient text, colored shadows, saturated backgrounds, bokeh, scroll-jacking, parallax, text animation, 3D transforms.
- **Reference Docs**: [Website Atmosphere](https://github.com/EverGrayTech/design-system/blob/main/docs/website-brand-atmosphere-and-graphics.md) · [Marketing Patterns](https://github.com/EverGrayTech/design-system/blob/main/docs/marketing-site-patterns.md)
